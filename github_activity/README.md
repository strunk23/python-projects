# Github Activity

A command line program to display user's github activity.

## Features
- Get most recent public activity from a given github user.
- Caching up to 5 user results to save API calls

## Usage

First clone the repo and go the github_activity directory

```bash
git clone https://github.com/strunk23/python-projects.git
cd python-projects/github_activity/
```

Get activity by typing:

```bash
python github_activity.py --user strunk23
```

Project based on [roadmap.sh](https://roadmap.sh/projects/github-user-activity)
